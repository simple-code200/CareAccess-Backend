\# Care Access Portal - Backend 🏥



\## 📖 Overview

This repository contains the backend APIs for the Care Access Portal.



The backend handles:

\- Authentication

\- Appointments

\- Patient Records

\- Doctor Data



\## ✨ Features

\- REST API Services

\- Role-Based Login

\- CRUD Operations

\- JSON Responses

\- CORS Enabled



\## 🛠️ Tech Stack

\- Python

\- Flask

\- Flask-CORS
